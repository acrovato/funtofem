transformEquivRigidMotion()¶

Compute least-squares fit of rigid rotation and translation to set of aerodynamic surface displacements. Use rotation and translation to compute difference between given displacements and the displacements due to rigid motion alone, i.e. the elastic deformation. Store the computed rotation, translation, and elastic deformation in empty input arrays

Parameters:
  • aero_disps (ndarray) – One-dimensional array of aerodynamic displacements
  • R ((3,3) ndarray) – Empty (3,3) array for output rotation matrix
  • t ((,3) ndarray) – Empty (3,1) array for output translatio
  • u (ndarray) – One-dimensional array of elastic deformations of size of aerodynamic displacements
