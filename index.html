Welcome to FUNtoFEM’s Documentation¶

If you are using FUNtoFEM with existing interface class and solvers, you should only need the FUNtoFEM Model and FUNtoFEM Driver. If you wish to create a new interface or a new driver, see the Programmer’s Guide.
